안녕하세요, 문제를 해결하는 백엔드 개발자를 꿈꾸는 김재민입니다

백엔드 개발을 중심으로 웹 서비스의 구조와 흐름을 이해하려 노력하고 있는 개발자입니다.
Spring Boot 기반의 서버 개발과 API 설계에 관심이 많으며, 기능 구현을 단계적으로 완성해 나가는 방식을 선호합니다.
꾸준한 학습과 개선을 통해 실제 서비스에 기여할 수 있는 개발자로 성장하고 싶습니다.

나를 한 문장으로 표현하면

“문제가 생기면 원인을 끝까지 파고들어 해결하려는 끈기 있는 개발자”입니다.

왜 웹 개발 수업을 신청하게 되었나요?

웹 개발은 프론트엔드와 백엔드가 유기적으로 연결되어 하나의 서비스를 완성한다는 점에서 큰 매력을 느꼈습니다.

특히 백엔드 개발자로서 서버와 데이터 흐름을 더 깊이 이해하고, 실제로 동작하는 웹 서비스를 직접 만들어 보고 싶어 수업을 신청하게 되었습니다.

HTML이란?

        HTML(HyperText Markup Language)은 웹페이지의 구조와 내용을 정의하는 마크업 언어로,
        텍스트, 이미지 등의 콘텐츠를 구조화하고 의미를 부여하며,
        CSS로 스타일을 꾸미고 JavaScript로 상호작용을 추가하는 웹의 기본 언어입니다.
        태그와 속성으로 구성되어 있으며, 
        웹 표준과 접근성을 위해 적절한 태그 사용이 중요합니다. 
        마크업 언어: 프로그래밍 언어가 아닌, 마크업(표시)을 통해 문서의 구조를 정의합니다.
        구조화: 제목, 문단, 목록, 이미지, 링크 등을 정의하여 웹페이지의 기본 뼈대를 만듭니다.
        웹의 기본 구성 요소: 웹 개발의 출발점이며, CSS(스타일)와 JavaScript(동작)와 함께 사용됩니다. 
        HTML의 구성 요소
        태그(Tag): <>로 감싸진 것으로, 요소의 시작과 끝을 나타내며 콘텐츠의 의미를 부여합니다.

CSS란?

        CSS(Cascading Style Sheets)는 HTML이나 XML로 작성된 웹 문서의 스타일(디자인, 레이아웃, 색상 등)을 정의하는 스타일 시트 언어로, 웹페이지의 구조(HTML)와 표현(CSS)을 분리하여 웹사이트를 꾸미고 일관된 디자인을 유지하게 합니다. HTML이 뼈대라면 CSS는 메이크업이나 옷처럼 웹페이지를 시각적으로 아름답게 만드는 역할을 하며, 요소의 위치, 글꼴, 색상 등을 지정할 수 있습니다. 
        주요 역할
        디자인 지정: 글자 크기, 색상, 배경 등을 설정합니다.
        레이아웃 제어: 웹페이지 요소들을 화면에 배치하고 정렬합니다 (예: 중앙 정렬, 좌우 배치).
        반응형 디자인: 다양한 기기(PC, 모바일)에 맞춰 레이아웃을 변경합니다.
        일관성 유지: 여러 페이지에 동일한 스타일을 쉽게 적용하여 일관된 디자인을 유지합니다.

JS란?

        1995년에 브랜든 아이크가 개발하고, 1996년에 정식 출시되며, 
        1997년에 ECMA International 소속의 TC39에서 표준화된 프로토타입 기반의 프로그래밍 언어로,스크립트 언어에 해당된다.
        특수한 목적이 아닌 이상 모든 웹 브라우저에 JavaScript 엔진이라는 인터프리터가 내장되어 있다.
        오늘날 HTML, CSS와 함께 웹을 구성하는 요소 중 하나다. HTML이 웹 페이지의 기본 구조를 담당하고,
        CSS가 디자인을 담당한다면 JavaScript는 클라이언트 단에서 웹 페이지가 동작하는 것을 담당한다.
        웹 페이지를 자동차에 비유하자면, HTML은 자동차의 뼈대,
        CSS는 자동차의 외관,
        JavaScript는 자동차의 동력원인 엔진이라고 볼 수 있다.
